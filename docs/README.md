## 🚀 Como Jogar

### 🏃‍♂️ Movimentação

- **W**, **A**, **S**, **D** ou as **Setas Direcionais**: Movimenta o personagem no mapa.

### ⚔️ Combate

- **Espaço**: Atacar inimigos.
- **Crtl**: Habilidades especiais (quando disponível).
- **Q** ou **e**: Trocar itens
